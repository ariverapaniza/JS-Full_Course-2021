    String Manipulation

    We use different string manipulation methods like: a.toUpperCase(), b.toLowerCase(), a.indexOf('H'), c.lastIndexOf('ello') [in case it repeats it will look for the last one], c.slice(6,16) [This will cut the parts of the string that we want], c.substr(3,3) [This will select the parts of the string that we want, the first number will be the starting point, the second number is how many characters we want to extract], c.substring(3,10) [with this one on the numbers we are adding the starting point and the ending point (this is the main difference between substr and substring)].
