    Nested Arrays

    Nested Arrays are the ones that will have an array inside an array. For example: var arr = [["cat", "dog"],[1,2,3,4], 40, 50]. It has 2 nested arrays inside the large array. To look for a nested item inside the array we use the method "arr[0][1]" 0 being the first nested array and 1 being the position of the item inside the nested array.
