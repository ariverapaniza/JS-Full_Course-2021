    The FOR loop is doing almost the same as the WHILE loop but the variables are inside the parenthesis. The form used is "for(var x = 0; x <= 31; x++){if(x % 2 !== 0){output1.innerHTML += "Value of x is = " +x+ ;};};". Variable X is inside and also the breaking point to make the condition (x <= 31) like a "while loop" and the x++ to sum up 1 each time it loops.
